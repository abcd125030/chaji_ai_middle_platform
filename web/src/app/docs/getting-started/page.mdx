# 快速开始

欢迎使用企业级AI中台平台！本指南将帮助您快速上手使用我们的平台。

## 概述

企业级AI中台平台是一个分布式架构的AI服务平台，提供：

- 🚀 **智能对话**：支持多轮对话和上下文理解
- 📚 **知识库管理**：向量数据库支持的智能检索
- 🔧 **工具集成**：丰富的AI工具和插件系统
- 🏗️ **工作流编排**：Graph-Node-Edge架构的任务编排
- 🛡️ **企业级安全**：完整的认证和权限管理

## 系统要求

### 基础环境
- Node.js 18+ 
- Python 3.9+
- PostgreSQL 13+
- Redis 6+

### 推荐配置
- CPU: 4核心以上
- 内存: 8GB以上
- 存储: 50GB以上可用空间

## 快速体验

### 1. 访问平台

打开浏览器访问：`https://your-domain.com`

### 2. 用户注册

点击右上角"注册"按钮，填写必要信息完成注册。

### 3. 开始对话

登录后，点击"开始聊天"，即可与AI助手进行对话。

## 核心功能

### 智能对话
```bash
# 示例对话
用户：你好，请帮我分析一下这份销售数据
AI：您好！我很乐意帮您分析销售数据。请您上传数据文件，我将为您提供详细的分析报告。
```

### 文件上传与处理
支持的文件格式：
- 📄 文档：`.docx`, `.pdf`, `.txt`
- 📊 表格：`.xlsx`, `.csv`
- 🖼️ 图片：`.png`, `.jpg`, `.jpeg`

### 知识库检索
平台内置智能检索功能，可以：
- 自动理解用户查询意图
- 从海量知识库中检索相关信息
- 提供准确的答案和引用来源
